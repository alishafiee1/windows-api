# راهنمای نصب و راه‌اندازی – سرویس محلی دسترسی به خط فرمان

---

## پیش‌نیازها

قبل از شروع، مطمئن شو این‌ها روی سیستمت هستند:

- **Python 3.10 یا بالاتر** — برای بررسی: `python --version`
- **PowerShell 5 یا بالاتر** — برای بررسی: `$PSVersionTable.PSVersion`
- **pip** — برای نصب وابستگی‌ها

---

## نصب

### ۱. دریافت فایل‌های پروژه

پوشه پروژه رو در یه مسیر دلخواه بساز، مثلاً:

```
C:\tools\local-cmd-api\
```

فایل‌های پروژه رو داخلش بریز.

### ۲. نصب وابستگی‌ها

یه بار توی PowerShell داخل پوشه پروژه این دستور رو بزن:

```powershell
pip install -r requirements.txt
```

### ۳. تنظیم فایل محیطی

فایل `env.example` رو کپی کن و اسمش رو به `.env` تغییر بده، بعد مقادیر رو طبق نیازت پر کن:

```
PORT=4001
BLOCKLIST_PATH=data/blocklist.txt
CONFIRM_TIMEOUT=120
SHELL=powershell
```

> **نکته:** `CONFIRM_TIMEOUT` یعنی کاربر چند ثانیه فرصت داره دستور خطرناک رو تأیید کنه. بعد از این مدت، توکن منقضی می‌شه.

---

## اجرا

### اجرای دستی

```powershell
python app.py
```

بعد از اجرا باید این رو ببینی:

```
* windows-cmd-api running on http://127.0.0.1:4001
* Open http://127.0.0.1:4001/help for usage guide
```

اگه سرور قبلی هنوز روی همون پورت بوده، خودکار بسته می‌شه:

```
* Stopped previous server process(es): 12345
```

### اجرای خودکار با ویندوز (اختیاری)

اگه می‌خوای سرویس با روشن شدن ویندوز خودکار بالا بیاد، یه Task در **Task Scheduler** بساز:

- **Trigger:** At startup
- **Action:** اجرای `python app.py` از مسیر پروژه
- **Run as:** کاربر فعلی

---

## تست اولیه

بعد از اجرا، این آدرس‌ها رو تست کن:

```
http://127.0.0.1:4001/          ← به /help هدایت می‌شه
http://127.0.0.1:4001/help      ← راهنمای کامل API (شامل بخش for_ai_assistants)
http://127.0.0.1:4001/run?cmd=Get-Date   ← تست سریع با GET
```

اگه JSON راهنما رو دیدی، سرویس سالمه.

## نحوه استفاده

### برای دستیار هوش مصنوعی (پیشنهادی)

اول `GET /help` رو بخون — فیلد `for_ai_assistants` و `examples` نمونه درخواست آماده داره.

### فرستادن دستور با POST (پیشنهادی)

```json
POST /run
Content-Type: application/json

{
  "cmd": "Get-Date"
}
```

```powershell
Invoke-RestMethod -Uri "http://127.0.0.1:4001/run" -Method POST `
  -ContentType "application/json" `
  -Body '{"cmd":"(Get-CimInstance Win32_OperatingSystem).Caption"}'
```

جواب:

```json
{
  "status": "ok",
  "output": "Microsoft Windows 11 Home",
  "error": null,
  "exit_code": 0,
  "duration": 0.42
}
```

### فرستادن دستور با GET (مرورگر / تست سریع)

```
http://127.0.0.1:4001/run?cmd=Get-Date
```

برای دستورات با فاصله یا کاراکتر خاص، مقدار `cmd` باید URL-encode بشه. برای دستورات پیچیده POST بهتره.

### وقتی دستور خطرناکه

اگه دستور توی blocklist باشه، جواب اینه:

```json
{
  "status": "pending",
  "token": "abc123",
  "confirm_url": "http://127.0.0.1:4001/confirm/abc123",
  "message": "این دستور نیاز به تأیید دستی دارد."
}
```

لینک `confirm_url` رو توی مرورگر باز کن — صفحه‌ای می‌بینی که دستور رو نشون می‌ده و دو دکمه داره: **تأیید** و **رد**.

---

## ویرایش لیست دستورات خطرناک

فایل `blocklist.txt` رو با هر ویرایشگر متنی باز کن. هر خط یه کلمه کلیدیه. اگه دستور ورودی شامل هر کدوم از این کلمات بشه، قبل از اجرا تأیید گرفته می‌شه.

**اضافه کردن قانون جدید:**

کافیه یه خط جدید اضافه کنی:

```
wipe
cipher /w
```

**حذف یه قانون:**

اون خط رو پاک کن و فایل رو ذخیره کن. نیازی به ریستارت سرویس نیست — فایل هر بار خونده می‌شه.

## رفع اشکال رایج

| مشکل | دلیل احتمالی | راه حل |
|------|--------------|--------|
| سرویس شروع نمی‌کنه | پورت قبلاً اشغاله | معمولاً با اجرای دوباره `python app.py` خودکار آزاد می‌شه؛ در غیر این صورت `PORT` رو در `.env` عوض کن |
| خطای 405 روی GET /run | سرور قدیمی در حافظه | همه `python app.py`‌ها رو ببند و فقط یکی اجرا کن |
| خطای `python not found` | Python توی PATH نیست | Python رو دوباره نصب کن و "Add to PATH" رو تیک بزن |
| صفحه تأیید باز نمی‌شه | توکن منقضی شده | دستور رو دوباره بفرست — توکن جدید می‌گیری |
| دستور اجرا نمی‌شه | PowerShell دسترسی نداره | سرویس رو با دسترسی Administrator اجرا کن |
| خروجی ناقصه | دستور طولانی بوده | خروجی از یه حد مشخص برش داده می‌شه — طبیعیه |

## متوقف کردن سرویس

در پنجره PowerShell که سرویس داره اجرا می‌شه:

```
Ctrl + C
```

فایل `data/server.pid` خودکار پاک می‌شه و پورت آزاد می‌شه.
